# 💼 Personal Portfolio

This is my personal portfolio website built with **React** and styled using **Tailwind CSS**. It showcases my projects, skills, and contact details in a clean, responsive design.

## 🚀 Tech Stack

- React (with Create React App)
- Tailwind CSS
- Lucide Icons
